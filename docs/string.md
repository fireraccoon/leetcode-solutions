## [28.找出字符串中第一个匹配项的下标](../src/28.找出字符串中第一个匹配项的下标.java)

KMP 算法。

关键是计算**前缀函数** $\pi(i)$，指的是长度为 $i$ 的前缀子串的最长相等**真前缀**和**真后缀**的长度，可利用**动态规划**进行求解。例如，字符串 `aabaaab` 的前缀函数为 $[0, 1, 0, 1, 2, 2, 3]$。

- 时间复杂度：$O(m+n)$

- 空间复杂度：$O(m)$
